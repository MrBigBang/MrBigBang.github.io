Android Studio如何导入第三方主题

Android Studio 是Google基于 IntelliJ IDEA 开发的，可以很方便的定义个性化的主题，下面我们来讨论下怎么导入第三方主题包吧。

下载第三放主题包

地址：http://www.ideacolorthemes.org/themes/

将主题包导入到Android Studio

打开Studio，依次按下面步骤进行导入
File—>Importing Settings—>找到刚下载好的主题jar包，点击确认即可。

对字体及颜色进行适当修改

  1. 进入到File->Settings->Editor->Colors&Fonts对字体和颜色进行适当的设置。
  2. 可能引入第三方主题后方法体左边可能会出现明显的竖线，去掉方法File->Settings->Editor->General->Appearance->将Show vertical indent guides选项取消即可将这些竖线隐藏掉。
